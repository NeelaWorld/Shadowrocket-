## 📦 Clash.MI 使用步骤（最像小火箭）

1. 打开 App  
2. 底部选择 **Profiles**  
3. 点击右上角 **＋ → Add URL**  
4. 粘贴订阅链接  
5. 保存后 **Set as Default**  
6. 首页开关 → 开启即可

---

## 📦 V2Box 使用步骤（新手最简单）

1. 打开 App  
2. 进入 **配置**  
3. 点击 **＋**  
4. 导入方式：URL / 扫码 / 文件  
5. 启用即可
