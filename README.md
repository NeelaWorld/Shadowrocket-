## 📦 Clash.MI 使用步骤（最像小火箭）

1. 打开 App  
2. 底部选择 **Profiles**  
3. 点击右上角 **＋ → Add URL**  
4. 粘贴订阅链接  
5. 保存后 **Set as Default**  
6. 首页开关 → 开启即可

## 📦 V2Box 使用步骤（新手最简单）

1. 打开 App  
2. 进入 **配置**  
3. 点击 **＋**  
4. 导入方式：URL / 扫码 / 文件  
5. 启用即可

## 🆚 一眼对比：应该选哪个？

三款工具定位清晰：  
- **ClashMI：最接近小火箭、无广告、免费，iOS 用户首选**  
- **V2Box：新手最简单、支持最多协议（iOS + Android）**  
- **Shadowrocket：功能最强，但付费且需海外 Apple ID**

| 对比项目 | ⭐ Clash.MI（免费） | ⭐ V2Box（免费） | ⭐ Shadowrocket（付费） |
|---------|---------------------|------------------|-------------------------|
| 平台支持 | iOS | iOS / Android | iOS（付费） |
| 是否免费 | ✔ 免费 | ✔ 免费 | ❌ 付费（需海外 Apple ID） |
| 广告情况 | ❌ 无广告 | ✔ 有广告 | ❌ 无广告 |
| 使用难度 | 中等（像小火箭） | 最简单（新手友好） | 中等偏高（功能多） |
| 协议支持 | SSR / SS / Hysteria / Trojan 等| SSR / SS / VMess / VLESS / Trojan 等| SSR / SS / Vmess / Trojan 等 |
| 是否类似小火箭 | ⭐⭐⭐⭐⭐ 最高 | ⭐⭐ 较像 | ⭐⭐⭐⭐ 本体 |
| Android 支持 | ❌ 无 | ✔ 有 | ❌ 无 |
| 配置灵活度 | 高 | 中 | 最高 |
| 适用人群 | 想找小火箭“免费平替”的 iOS 用户 | 想简单一键导入的新手 | 老玩家 / 想要完整功能的人 |

### 📌 结论（建议选择）：

- **iOS 用户推荐：Clash.MI（免费 + 无广告 + 最像小火箭）**  
- **需求最强功能：Shadowrocket（付费）**
